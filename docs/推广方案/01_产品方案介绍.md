# 企查查MCP金融插件产品方案

### 1.1 产品定位

### 1.2 核心价值主张

> **让金融机构尽调从"人工低效"走向"智能秒批"**

| 2-3小时人工查询 | 30秒自动核验 | **效率提升99%** |
| 多平台数据分散 | 65个工具一键调用 | **数据全覆盖** |

## 三、产品功能

### 3.1 KYB自动化核验 (Know Your Business)

**适用场景**: 银行开户、信贷审批、反洗钱合规、企业年检

**核心能力**:

| 功能模块 | 详细说明 | MCP工具 |
|---------|---------|---------|
| **实体锚定** | 统一社会信用代码与企业名称交叉验证 | get_company_registration_info |
| **工商核验** | 注册资本、法定代表人、股东结构实时比对 | get_shareholder_info |
| **受益所有人识别** | 股权穿透自动识别实控人(UBO) | get_shareholder_info + 穿透算法 |
| **18类风险扫描** | 失信/被执行/限高/破产/股权冻结等 | 7个核心风险工具 |
| **KYB评级** | A/B/C/D四级自动评级+准入建议 | 综合评估算法 |
| **关联关系排查** | 集团客户、隐性关联识别 | get_external_investments |

**输出报告**:
```
================================================================
KYB核验报告
================================================================
核验企业: 华为技术有限公司
统一社会信用代码: 9144030019238XXXX
核验状态: 通过 ✓
整体风险: LOW (A级)
核验耗时: 28秒

核验结果:
  ✓ 企业状态: 存续
  ✓ 法定代表人: 匹配
  ✓ 注册资本: 匹配
  ✓ 风险扫描: 0项CRITICAL, 0项HIGH

准入建议: 正常准入。主体合法存续，无明显风险信号。
================================================================
```

### 3.2 IC Memo全维度背调 (Investment Committee Memorandum)

**适用场景**: PE/VC投资尽调、并购重组、战略投资、定增认购

**7章标准结构**:

| 章节 | 内容 | MCP工具数 |
|------|------|----------|
| **Chapter 1** | 执行摘要（投资建议+核心亮点+关键风险） | - |
| **Chapter 2** | 公司概况与股权结构（含受益所有人） | 4个 |
| **Chapter 3** | 知识产权与核心竞争力 | 5个 |
| **Chapter 4** | 法律与合规风险（全量司法风险扫描） | 16个 |
| **Chapter 5** | 经营与市场分析（招投标+舆情） | 7个 |
| **Chapter 6** | 财务分析 | 1个 |
| **Chapter 7** | 投资建议与风险提示 | - |

**知识产权尽调**:
- 专利布局分析（发明/实用新型/外观）
- 商标品牌核查
- 软件著作权统计
- 核心技术评估

**司法风险扫描**:
- 涉诉情况（原告/被告/被执行人）
- 失信信息
- 限制高消费
- 股权冻结/质押
- 破产重整
- 终本案件

**经营动态分析**:
- 招投标活跃度（业务规模指标）
- 资质证书有效性
- 信用评级
- 监管抽查结果
- 新闻舆情监控

## 四、应用场景

### 4.1 银行对公业务

**场景**: 企业客户开户KYB

**痛点**:
- 客户经理手动查询工商信息，耗时2-3小时/户
- 风险排查不全面，容易遗漏关联风险
- 反洗钱合规压力大

**解决方案**:
```
客户经理输入: /kyb-verification-qcc 深圳市XX科技有限公司

30秒后获得:
├── 工商信息核验（自动比对申请表）
├── 18类风险扫描（CRITICAL/HIGH/MEDIUM）
├── 受益所有人识别（反洗钱合规）
├── 关联企业排查（集团客户识别）
└── KYB评级+准入建议（A/B/C/D）
```

**效果**:
- 尽调时间: 2-3小时 → 30秒
- 人工成本: 降低90%
- 风险遗漏: 降低95%

---

### 4.2 PE/VC投资尽调

**场景**: Pre-IPO项目投资决策

**痛点**:
- 尽调周期长（1-2周）
- 多平台数据收集繁琐
- IC Memo编写耗时

**解决方案**:
```
投资经理输入: /ic-memo-qcc 北京XX科技有限公司 --sector 人工智能

自动生成:
├── Chapter 1: 执行摘要（投资建议）
├── Chapter 2: 公司概况（股权穿透+实控人）
├── Chapter 3: 知识产权（50项发明专利清单）
├── Chapter 4: 法律风险（3项诉讼详细分析）
├── Chapter 5: 经营分析（近12个月招投标12次）
├── Chapter 6: 财务分析（近3年营收趋势）
└── Chapter 7: 投资建议（推荐+风险提示）
```

**效果**:
- 尽调周期: 1-2周 → 1天
- 报告编写: 2天 → 自动填充
- 数据覆盖率: 提升200%

---

### 4.3 企业战投

**场景**: 战略投资标的筛选

**痛点**:
- 标的数量多，尽调资源有限
- 需要快速排除高风险标的
- 行业对标分析困难

**解决方案**:
```
批量尽调: /ic-memo-qcc 标的清单.xlsx

输出:
├── 标的优先级排序（风险评分）
├── 行业对比分析（市场份额）
├── 技术壁垒评估（专利布局）
└── 投资建议汇总（通过/否决/待定）
```
