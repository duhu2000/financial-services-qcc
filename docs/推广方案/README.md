## 快速使用指南

### 场景1：内部产品培训
**推荐文档**：01_产品方案介绍.md
**使用方法**：
- 作为产品培训教材
- 了解产品定位、核心功能、竞争优势
- 掌握KYB和IC Memo两大核心方案

### 场景2：技术社区推广
**推荐文档**：02_技术博客文章.md
**使用方法**：
- 发布到知乎、掘金、InfoQ等技术社区
- 文章一：技术深度解析篇（适合技术人员）
- 文章二：业务场景实践篇（适合业务人员）
- 文章三：行业趋势洞察篇（适合管理层）

### 场景3：客户案例宣传
**推荐文档**：03_业务场景软文.md
**使用方法**：
- 软文一：银行对公业务篇 → 发布到银行家杂志、金融时报
- 软文二：PE/VC投资尽调篇 → 发布到36氪、投中网
- 软文三：企业战投篇 → 发布到中国企业家、哈佛商业评论
- 软文四：反洗钱合规篇 → 发布到反洗钱实务、金融监管

### 场景4：销售获客
**推荐文档**：04_客户搜索策略.md
**使用方法**：
- 目标客户清单：银行、PE/VC、企业战投
- 获客渠道：线上（内容/SEO/社群）+ 线下（展会/沙龙）
- 销售漏斗：从潜在客户到续约客户的转化路径
- 定价策略：基础版/专业版/企业版

### 场景5：行业会议演讲
**推荐文档**：05_行业会议PPT框架.md
**使用方法**：
- PPT一：金融科技峰会演讲版（30-40分钟）
  - 适用：中国国际金融展、银行家年会
  - 结构：痛点→方案→案例→展望
- PPT二：投资峰会路演版（15-20分钟）
  - 适用：投中年会、36氪WISE大会
  - 结构：痛点→方案→案例→价值

### 场景6：商务洽谈/招投标
**推荐文档**：06_Word产品方案介绍.md
**使用方法**：
- 导出为Word/PDF格式
- 作为商务洽谈的标准文档
- 用于招投标投标文件
- 包含完整的产品功能、案例、商务方案

## 核心卖点提炼

### 一句话价值主张
> **"让金融机构尽调从人工低效走向智能秒批"**

### 三大核心价值
1. **效率革命**：30秒完成传统2-3小时的尽调工作（提升99%）
2. **全面扫描**：18类风险实时扫描，65个MCP工具一键调用
3. **智能分析**：AI自动评级，A/B/C/D四级准入建议

| 指标 | 传统方式 | MCP方式 | 提升 |
|------|----------|---------|------|

| 风险遗漏率 | 12-15% | 0.3% | **98%** |
| 数据覆盖 | 5-8类 | 18类 | **225%** |
