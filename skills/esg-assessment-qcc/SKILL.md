# ESG风险评估 (ESG Assessment) v1.0

## 技能概述

这个Skill用于**企业ESG（环境、社会、治理）风险评估**，专为ESG投资机构、上市公司ESG报告编制、绿色金融产品设计而打造。

**核心价值**：
- **数据自动化**：企查查MCP自动采集E/S/G三维度数据
- **评分标准化**：基于国际ESG评级框架的企业评分
- **报告智能化**：自动生成ESG评估报告
- **投资辅助**：支持可持续投资决策

**适用场景**：
- ESG主题投资基金尽调
- 上市公司ESG评级
- 绿色债券发行评估
- 供应链金融ESG准入
- 可持续投资筛选

## ESG评分模型

### 三维度评分

```
ESG综合评分
├── E-环境 (Environmental) 权重 35%
│   ├── 环保合规 40%
│   │   └── 企查查：环保处罚、排污许可
│   ├── 环境管理 35%
│   │   └── 需人工补充：环保投入、ISO14001
│   └── 碳排放 25%
│       └── 需人工补充：碳排放数据、减排措施
├── S-社会 (Social) 权重 35%
│   ├── 员工权益 40%
│   │   └── 企查查：劳动仲裁 + 需人工补充：社保
│   ├── 社会责任 35%
│   │   └── 需人工补充：公益捐赠、社区关系
│   └── 舆情声誉 25%
│       └── 企查查：新闻舆情情感分析
└── G-治理 (Governance) 权重 30%
    ├── 公司治理 40%
    │   └── 企查查：股权结构、实控人稳定性
    ├── 合规治理 35%
    │   └── 企查查：司法诉讼、行政处罚、异常
    └── 信息透明 25%
        └── 企查查：工商信息完整性
```

### 评分等级

| 等级 | 分数 | 说明 | 投资建议 |
|------|------|------|----------|
| AAA | 90-100 | ESG表现卓越 | 强烈推荐 |
| AA | 80-89 | ESG表现优秀 | 推荐 |
| A | 70-79 | ESG表现良好 | 可考虑 |
| BBB | 60-69 | ESG表现一般 | 谨慎考虑 |
| BB | 50-59 | ESG表现较弱 | 高风险 |
| B | 0-49 | ESG表现差 | 回避 |

## 标准工作流

### 第一步：企业识别与数据采集

```
企查查MCP数据采集流程：
┌─────────────────────────────────────────┐
│  输入：企业全称                          │
└──────────────┬──────────────────────────┘
               │
    ┌──────────┼──────────┬──────────┐
    │          │          │          │
┌───▼───┐ ┌───▼───┐ ┌───▼───┐ ┌───▼───┐
│ E-环境 │ │ S-社会 │ │ G-治理 │ │ 舆情  │
│ 数据   │ │ 数据   │ │ 数据   │ │ 分析  │
└───┬───┘ └───┬───┘ └───┬───┘ └───┬───┘
    │         │         │         │
    └─────────┴─────────┴─────────┘
              │
         ┌────▼────┐
         │ESG评分  │
         │计算     │
         └─────────┘
```

### 第二步：三维度评分计算

**E-环境评分**（企查查自动）
- 环保处罚：无处罚得满分，有处罚扣分
- 排污许可：有许可得分，无许可扣分

**S-社会评分**（企查查+人工）
- 劳动仲裁：无仲裁得满分
- 舆情情感：正面舆情加分，负面扣分

**G-治理评分**（企查查自动）
- 股权稳定性：稳定得分，频繁变更扣分
- 合规记录：无违规得满分

### 第三步：生成ESG报告

整合三维度评分，输出：
- ESG综合评分与等级
- 三维度分项评分
- 行业对标分析
- 改进建议
