# 关联方穿透识别 (Related Party Identification) v1.0

## 技能概述

这个Skill用于**企业关联方自动识别与穿透分析**，专为投资银行IPO项目、并购交易、审计合规、风控管理设计。

**核心价值**：
- **股权穿透**：自动识别实际控制人、最终受益人
- **关联图谱**：可视化展示企业关联关系网络
- **关联交易**：识别潜在关联交易与利益输送
- **一致行动人**：发现隐性一致行动人关系

**适用场景**：
- IPO关联方核查
- 并购关联交易审查
- 集团客户识别
- 银行授信集中度管理
- 合规利益冲突检测

## 企查查MCP增强说明

### 关联方数据采集

| 关联方类型 | 识别方式 | MCP Server | 输出内容 |
|------------|----------|------------|----------|
| **股东** | 股权结构查询 | `qcc-company` | 直接股东、持股比例 |
| **实控人** | 股权穿透计算 | `qcc-company` | 自然人/国有/外资实控人 |
| **对外投资** | 子公司识别 | `qcc-company` | 持股企业、投资比例 |
| **分支机构** | 分公司/代表处 | `qcc-company` | 分支机构清单 |
| **董监高** | 关键人员 | `qcc-company` | 法定代表人、高管 |
| **历史变更** | 股权变更 | `qcc-company` | 历次股权变动 |

### 穿透层级

- **标准穿透**：向上3层（股东→股东之股东→最终实控人）
- **深度穿透**：向上5层（适用于复杂股权结构）
- **全量穿透**：无限制穿透（适用于IPO核查）

## 关联方识别范围

### 标准关联方定义（基于IPO标准）

```
关联方识别范围
├── 一、股权关联方
│   ├── 直接/间接持股5%以上股东
│   ├── 实际控制人及其近亲属
│   ├── 控股股东及其控制的其他企业
│   └── 一致行动人及其控制企业
├── 二、人员关联方
│   ├── 董监高及其近亲属
│   ├── 核心技术人员及其近亲属
│   └── 上述人员控制或担任高管的企业
├── 三、交易关联方
│   ├── 报告期内重大交易对手
│   ├── 疑似关联交易（价格异常）
│   └── 资金往来异常企业
└── 四、其他关联方
    ├── 同一控制下企业
    ├── 重大影响企业（联营/合营）
    └── 潜在关联方（需进一步核查）
```

## 标准工作流

### 第一步：股权穿透分析

```
企查查MCP股权穿透流程：
┌─────────────────────────────────────────┐
│  输入：目标企业全称                      │
└──────────────┬──────────────────────────┘
               │
        ┌──────▼──────┐
        │ 查询股权结构 │
        │ qcc-company │
        └──────┬──────┘
               │
    ┌──────────┼──────────┐
    │          │          │
┌───▼───┐ ┌───▼───┐ ┌───▼───┐
│第一层 │ │第二层 │ │第三层 │
│股东   │ │股东   │ │股东   │
└───┬───┘ └───┬───┘ └───┬───┘
    │         │         │
    └─────────┴─────────┘
              │
         ┌────▼────┐
         │实控人识别│
         │关联图谱  │
         └─────────┘
```

### 第二步：实控人计算

**穿透计算规则**：
1. 逐层向上追溯股权比例
2. 计算综合持股比例（乘法）
3. 识别持股比例最高的自然人/实体
4. 判断是否为实际控制人（通常>30%）

### 第三步：关联图谱生成

整合股权、人员、交易数据，生成：
- 股权结构图
- 关联关系网络图
- 关联方清单

### 第四步：关联交易识别

基于企查查数据识别：
- 报告期内重大交易对手
- 疑似关联交易（交易对手为关联方）
- 资金往来异常

## 穿透计算规则

### 综合持股比例计算

```
示例计算：

张三 → 北京投资(80%) → 目标企业(20%)
     → 上海投资(70%) → 目标企业(15%)
     → 直接持股 → 目标企业(35%)

张三综合持股：
= 35%(直接) + 80%×20%(间接) + 70%×15%(间接)
= 35% + 16% + 10.5%
= 61.5%

结论：张三为实际控制人（持股>30%）
```

### 一致行动人认定

```
认定依据：
1. 股权控制关系（同一实控人）
2. 亲属关系（夫妻、父母子女）
3. 协议安排（一致行动协议）
4. 历史投票记录（一致表决）
5. 资金关联（共同出资、担保）
```

### 示例1：IPO关联方核查

**用户输入**：
> "对拟IPO企业进行关联方穿透识别，用于招股书披露"

**Skill执行**：

**用户输入**：
> "识别某集团的全部关联企业，用于银行授信集中度管理"

**Skill执行**：
1. 从集团母公司开始穿透
2. 识别所有子公司、参股公司
3. 识别同一控制下其他企业
4. 生成集团客户清单

**输出**：
- 集团母公司：XX集团
- 纳入集团客户管理：共15家企业
