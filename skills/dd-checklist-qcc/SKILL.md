# 尽职调查清单自动化 (Due Diligence Checklist) v1.0

## 技能概述

这个Skill用于**自动化管理尽职调查全流程**，专为PE/VC投资机构、投资银行、并购团队设计。

**核心价值**：
- **清单标准化**：自动生成法律、财务、业务、技术四大尽调清单
- **数据自动化**：企查查MCP实时填充可获取的数据项
- **进度可视化**：追踪尽调任务完成状态
- **报告智能化**：自动生成尽调报告底稿

**适用场景**：
- PE/VC投资尽调
- 并购交易尽调
- IPO尽职调查
- 债券发行尽调
- 战略投资评估

### 数据填充标识

清单中的数据项会标注来源：
- 🤖 **企查查自动填充**：已由MCP获取并填充
- 📝 **需人工补充**：需尽调团队人工收集
- 🔗 **可交叉验证**：企查查数据与人工收集数据比对

- **天使/种子轮**：精简版尽调清单（法律+业务）
- **A/B轮**：标准版尽调清单（法律+财务+业务）

### Level 1: 法律尽职调查 (Legal DD)

```
法律尽调清单
├── 1. 主体资格与公司治理

│   ├── 1.3 历史沿革 🤖
│   │   └── 企查查自动填充：历次变更记录
│   └── 1.4 公司治理 📝
│       └── 需人工补充：章程、议事规则、董事会决议
├── 2. 资产与知识产权
│   ├── 2.1 知识产权清单 🤖
│   │   └── 企查查自动填充：专利、商标、软著
│   └── 2.2 无形资产 📝
│       └── 需人工补充：专有技术、域名、著作权
├── 3. 重大合同与债权债务
│   ├── 3.1 重大合同 📝
│   │   └── 需人工补充：前十大合同、关联交易
│   └── 3.2 担保与抵押 📝
│       └── 需人工补充：担保合同、抵押登记
├── 4. 诉讼与合规
│   ├── 4.1 司法诉讼 🤖
│   │   └── 企查查自动填充：立案信息、判决文书
│   ├── 4.2 失信记录 🤖
│   │   └── 企查查自动填充：失信被执行人
│   ├── 4.3 被执行信息 🤖
│   │   └── 企查查自动填充：被执行人记录
│   ├── 4.4 行政处罚 🤖
│   │   └── 企查查自动填充：处罚记录
│   └── 4.5 经营异常 🤖
│       └── 企查查自动填充：异常名录记录
└── 5. 劳动用工
    └── 📝 需人工补充：劳动合同、社保缴纳
```

## 标准工作流

### 第一阶段：初始化尽调项目

**输入**：
- 目标企业名称
- 投资阶段（天使/A/B/C/Pre-IPO/并购）
- 尽调范围（法律/财务/业务/技术）

**Skill执行**：
1. 识别企业全称
2. 调用企查查MCP获取基础数据
3. 生成标准化尽调清单
4. 自动填充可获取的数据项

### 第二阶段：数据自动化填充

```
企查查MCP批量调用：
┌─────────────────────────────────────────┐
│  输入：目标企业全称 + 尽调范围           │
└──────────────┬──────────────────────────┘
               │
    ┌──────────┼──────────┬──────────┐
    │          │          │          │
┌───▼───┐ ┌───▼───┐ ┌───▼───┐ ┌───▼───┐
│ 法律  │ │ 财务  │ │ 业务  │ │ 技术  │
│ 数据  │ │ 数据  │ │ 数据  │ │ 数据  │
└───┬───┘ └───┬───┘ └───┬───┘ └───┬───┘
    │         │         │         │
    └─────────┴─────────┴─────────┘
              │
         ┌────▼────┐
         │自动填充 │
         │尽调清单 │
         └─────────┘
```

### 第三阶段：进度追踪与更新

- 标记已完成的任务项
- 更新企查查实时数据（风险变化自动提醒）
- 生成进度报告

### 第四阶段：生成尽调报告

整合已完成的尽调内容，输出：
- 法律尽调报告
- 财务尽调报告
- 业务尽调报告
- 技术尽调报告（如适用）
- 综合尽调摘要

## 使用示例

### 示例：启动A轮投资尽调

**用户输入**：
> "启动对北京某某科技有限公司的A轮投资尽调，需要法律、财务、业务三个维度"
