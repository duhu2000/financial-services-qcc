## MCP 配置要求

**⚠️ 重要：使用本SKILL前，必须确保企查查MCP服务器已配置**

### 检查清单：
1. ✅ `~/.claude/.mcp.json` 文件存在且配置正确
2. ✅ `QCC_MCP_API_KEY` 环境变量已设置
3. ✅ Claude Code 已重启加载MCP配置

### 配置方法：
```bash
# 1. 创建 MCP 配置文件
cat > ~/.claude/.mcp.json << 'EOF'
{
  "mcpServers": {
    "qcc-ipr": {
      "url": "https://agent.qcc.com/mcp/ipr/stream",
      "headers": { "Authorization": "Bearer ${QCC_MCP_API_KEY}" }
    },
    "qcc-company": {
      "url": "https://agent.qcc.com/mcp/company/stream",
      "headers": { "Authorization": "Bearer ${QCC_MCP_API_KEY}" }
    },
    "qcc-risk": {
      "url": "https://agent.qcc.com/mcp/risk/stream",
      "headers": { "Authorization": "Bearer ${QCC_MCP_API_KEY}" }
    }
  }
}
EOF

# 2. 设置 API Key
export QCC_MCP_API_KEY="your_api_key_here"

# 3. 重启 Claude Code
```

详见文档：https://github.com/duhu2000/financial-services-qcc/blob/main/docs/MCP_CONFIGURATION.md

## UNIVERSAL RULES (适用于所有知产尽调任务)

- **NEVER** 仅统计专利数量——必须分析专利质量、技术领域分布、法律状态
- **NEVER** 忽视专利的有效性——必须核查是否存在无效、届满、终止情况
- **NEVER** 对核心技术不评估侵权风险——FTO（自由实施）分析至关重要
- **NEVER** 忽略知识产权的权属瑕疵——职务发明、合作开发、委托开发需核查
- **ALWAYS** 标注专利的法律状态：有效/无效/在审/届满
- **ALWAYS** 评估知识产权与主营业务的匹配度
- **ALWAYS** 核查知识产权质押、转让、许可情况
- **FOR CHINESE ENTERPRISES: ALWAYS use QCC MCP as primary data source**

## 知产尽调工作流 (4大模块)

### Module 1: 专利全景扫描 (Patent Portfolio Analysis)

**目标**: 全面盘点企业专利资产，评估技术实力

**企查查MCP调用**:
1. **专利列表查询** (qcc_ipr/get_patent_list)
   - 获取全部专利清单
   - 分类统计：发明/实用新型/外观设计

2. **专利详情查询** (qcc_ipr/get_patent_details)

### Module 4: 知产风险与权属核查 (IP Risk & Ownership)

1. **裁判文书查询** (qcc_risk/get_court_judgments)
   - 专利/商标侵权诉讼
   - 技术合同纠纷

2. **开庭公告查询** (qcc_risk/get_court_announcements)
   - 知产类案件开庭信息

**企查查MCP调用** (qcc_ipr Server):
3. **专利质押查询** (qcc_ipr/get_patent_pledge)
   - 专利质押融资情况

4. **专利转让查询** (qcc_ipr/get_patent_transfer)

## NEVER DO THESE

- NEVER 仅统计专利数量而不分析质量与有效性
- NEVER 对核心技术不进行侵权风险分析
- NEVER 忽视职务发明权属核查
- NEVER 对即将到期的核心知识产权不进行提示
- NEVER 不核查知识产权质押情况
- NEVER 忽略知识产权诉讼的潜在赔偿风险
- NEVER 对知产密集型企业的尽调报告不经过IP律师审核

ALL OUTPUTS REQUIRE REVIEW BY A QUALIFIED IP ATTORNEY BEFORE USE IN INVESTMENT DECISIONS.
