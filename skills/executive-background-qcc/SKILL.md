## MCP 配置要求

**⚠️ 重要：使用本SKILL前，必须确保企查查MCP服务器已配置**

### 检查清单：
1. ✅ `~/.claude/.mcp.json` 文件存在且配置正确
2. ✅ `QCC_MCP_API_KEY` 环境变量已设置
3. ✅ Claude Code 已重启加载MCP配置

### 配置方法：
```bash
# 1. 创建 MCP 配置文件
cat > ~/.claude/.mcp.json << 'EOF'
{
  "mcpServers": {
    "qcc-company": {
      "url": "https://agent.qcc.com/mcp/company/stream",
      "headers": { "Authorization": "Bearer ${QCC_MCP_API_KEY}" }
    },
    "qcc-risk": {
      "url": "https://agent.qcc.com/mcp/risk/stream",
      "headers": { "Authorization": "Bearer ${QCC_MCP_API_KEY}" }
    }
  }
}
EOF

# 2. 设置 API Key
export QCC_MCP_API_KEY="your_api_key_here"

# 3. 重启 Claude Code
```

详见文档：https://github.com/duhu2000/financial-services-qcc/blob/main/docs/MCP_CONFIGURATION.md

## UNIVERSAL RULES (适用于所有高管背调任务)

- **NEVER** 仅核查现任职务——必须追溯历史任职记录
- **NEVER** 忽视高管的个人司法风险——个人失信可能影响企业
- **NEVER** 对关联企业不进行风险扫描
- **NEVER** 忽略竞业禁止与利益冲突排查
- **ALWAYS** 区分法定代表人/董事长/总经理的不同风险点
- **ALWAYS** 核查高管是否为多家企业的实控人
- **ALWAYS** 标注数据的时效性与局限性
- **FOR CHINESE ENTERPRISES: ALWAYS use QCC MCP as primary data source**

## 高管背调工作流 (5维度)

### Dimension 1: 高管身份与任职核验 (Identity Verification)

**目标**: 核验高管身份信息与任职记录

**企查查MCP调用**:
1. **主要人员查询** (qcc_company/get_key_personnel)
   - 董事、监事、高管名单
   - 职务与任期

2. **法定代表人信息** (qcc_company/get_company_registration_info)
   - 法人姓名、证件号码

**高管清单**:

## 高管背调综合评级

### 评级体系

```
高管背调评级: [A/B/C/D]

评级标准 (按高管个人):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
评级    标准                                    建议
─────────────────────────────────────────────────────
A级     无个人风险，履历清晰，                  可正常聘任
        无利益冲突
B级     无重大个人风险，履历有 minor 瑕疵，     可聘任，加强监测
        或存在可解释的利益关系
C级     有个人风险记录但已解决，                谨慎聘任，需说明
        或存在潜在利益冲突
D级     当前失信被执行人，                      建议不聘任
        或存在重大利益冲突

整体评级: [A/B/C/D] (取最低分项)
```

## NEVER DO THESE

- NEVER 仅核查现任职务而忽视历史记录
- NEVER 对高管个人失信记录不进行标注
- NEVER 忽视关联企业的风险传导
- NEVER 对竞业禁止风险不进行提示
- NEVER 将个人风险与企业风险混为一谈
- NEVER 对关联交易嫌疑不进行排查
- NEVER 忽视高管频繁跳槽的风险信号
- NEVER 对高风险高管不给出明确的聘任建议

ALL OUTPUTS REQUIRE REVIEW BY A QUALIFIED HR OR LEGAL SPECIALIST BEFORE USE IN HIRING OR INVESTMENT DECISIONS.
