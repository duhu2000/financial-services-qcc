# 企业画像速览 (Strip Profile) v1.0 - 企查查MCP增强版

## 技能概述

这个Skill用于**快速生成一页纸企业画像**，专为投资银行、PE/VC投资机构、券商研究所设计。

通过企查查MCP四大Server实时获取数据，3分钟内输出包含以下维度的标准化企业速览：
- 企业基本信息与股权结构
- 18类风险信号扫描
- 知识产权资产盘点
- 经营动态与资质能力

**输出形式**：一页纸Markdown摘要 + 可选PPT格式 + 可选Word报告

---

## 企查查MCP增强说明

### 实时数据调用

本Skill自动调用企查查四大MCP Server获取实时数据：

| 数据维度 | MCP Server | 输出内容 |
|----------|------------|----------|
| 工商登记 | `qcc-company` | 注册资本、成立日期、法定代表人、经营状态 |
| 股权结构 | `qcc-company` | 股东构成、实控人、股权穿透、对外投资 |
| 风险扫描 | `qcc-risk` | 失信、被执行、限高、诉讼、异常、处罚 |
| 知产资产 | `qcc-ipr` | 专利、商标、软著、作品著作权 |
| 经营能力 | `qcc-operation` | 资质证书、招投标、融资记录、新闻舆情 |

### 数据更新机制

- **实时数据**：工商、风险、知产数据实时调用
- **动态摘要**：经营数据智能脱水，节省Token消耗
- **一键刷新**：支持"刷新数据"指令重新获取最新信息

## 标准工作流

### 第一步：接收企业名称

用户提供企业名称（全称或简称），Skill自动：
1. 识别企业全称
2. 调用`qcc-company`获取基础工商信息
3. 验证企业存续状态

### 第二步：并行获取多维度数据

```
企查查MCP并行调用流程：
┌─────────────────────────────────────────┐
│  输入：企业全称                          │
└──────────────┬──────────────────────────┘
               │
    ┌──────────┼──────────┬──────────┐
    │          │          │          │
┌───▼───┐ ┌───▼───┐ ┌───▼───┐ ┌───▼───┐
│qcc-   │ │qcc-   │ │qcc-   │ │qcc-   │
│company│ │risk   │ │ipr    │ │operation│
└───┬───┘ └───┬───┘ └───┬───┘ └───┬───┘
    │         │         │         │
    └─────────┴─────────┴─────────┘
              │
         ┌────▼────┐
         │数据整合  │
         │与结构化  │
         └────┬────┘
              │
    ┌──────────▼──────────┐
    │  输出：一页纸企业画像  │
    └─────────────────────┘
```

### 第三步：生成一页纸画像

按照标准模板整合数据，输出包含以下章节：

1. **企业名片**（基础信息）
2. **股权结构**（股东与实控人）
3. **风险扫描**（18类风险摘要）
4. **知产资产**（专利/商标/软著）
5. **经营动态**（资质/招投标/舆情）
6. **初步判断**（投资/合作建议）

## 使用示例

### 示例1：快速了解潜在投资标的

**用户输入**：
> "快速了解一下字节跳动这家公司，考虑投资机会"

**Skill执行**：
1. 调用`qcc-company`获取字节跳动工商信息
2. 调用`qcc-risk`扫描风险信号
3. 调用`qcc-ipr`获取知识产权数据
4. 调用`qcc-operation`获取经营动态
5. 生成一页纸企业画像

**输出**：
- 企业名片：成立日期、注册资本、法定代表人
- 股权结构：张一鸣持股XX%，股权穿透结果
- 风险扫描：🟢 无重大风险信号
- 知产资产：专利X项、商标X项
- 经营动态：最新融资、业务板块
- 初步判断：综合评级A（推荐关注）

### 示例2：竞品对比分析

**用户输入**：
> "对比一下美团和饿了么的企业画像"

**Skill执行**：
1. 分别获取两家公司数据
2. 生成对比表格
3. 输出横向分析

## 限制与注意事项

1. **数据覆盖**：部分敏感企业数据可能受限
2. **实时性**：风险数据T+1更新，非实时
3. **仅供参考**：Strip Profile为初步筛选工具，不构成投资建议
4. **隐私合规**：使用数据需遵守企查查服务协议
